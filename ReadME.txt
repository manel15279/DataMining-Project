Requirements:
gradio
squarify
psutil
mpl_toolkits
Delaunay 
seaborn

Execution:
1. Installez les librairies nécessaires a l'aide de pip
2. Ouvrez le fichier IHM.py
3. Executez le fichier dans un terminal
4. Clickez sur le lien généré en terminal pour lancer la page de l'application

Ps: - L'execution de DBSCAN peut donner des erreurs en calculant silhouette si les parametres ne sont pas bien choisis (il est cependant conseillé de suivre les parametres par défaut).
- Pour le clustering, il est nécessaire de réexecuter le fichier IHM.py pour chaque model.
